<실습> Microsoft Visual Studio Community 2022를 설치해 보자 

2022.03.06 초안 작성 

1. 우선 개발툴로 사용할 Microsoft Visual Studio Community 2022 설치 파일을 다운로드 하자.  

1-1.  웹  브라우저로 

https://visualstudio.microsoft.com/ko/vs/  또는  책의  사이트

(https://cafe.naver.com/thisisLinux)에서 Visual Studio Community 2022(vs_community.exe)를 다운로드 하자. 

(파일 이름 뒤에 버전 번호가 길게 붙을 수 있는데, 그냥 무시하고 사용하면 된다) 

[그림 1] Visual Studio Community 2022 다운로드  

1-2. 다운로드한 vs_community.exe를 실행해서, [Visual Studio Installer]에 서 <계속>을 클릭한다. 

[그림 2] Visual Studio Community 설치 1  

1-3. [설치] 창에서 'ASP.NET 및 웹 개발'을 선택하고, 오른쪽 아래의 '추가 프로젝트 템플릿(이전 버전)'도 추

가로 선택한 후에 <설치>를 클릭한다.  

 
 
 
 
 
 
 
 
 
 
[그림 3] Visual Studio Community 설치 2  

1-4. 한동안 설치가 진행된다. 컴퓨터 성능에 따라서 몇 십 분이 걸릴 수도 있다. 

[그림 4] Visual Studio Community 설치 3  

1-5. 설치가 완료되면 오른쪽 위 <x>를 클릭해서 설치를 마친다.  

[그림 5] Visual Studio Community 설치 4  

 
 
 
 
 
 
1-6. [Visual Studio Installer]도 종료하고 재부팅한다. 재부팅하지 않으면 추후 문제가 생길 수 있다. 

2. ASP.NET 웹 응용프로그램을 작성하자. 

2-1. Windows의 [시작] >> [Visual Studio 2022]를 클릭해서 실행하자. 

[그림 6] Visual Studio Community 2022 실행 1  

2-2. Visual Studio 시작 화면에서 '나중에 로그인'을 클릭한다. 

[그림 7] Visual Studio Community 2022 실행 2  

2-3 [친숙한 환경에서 시작] 화면에서 적당한 색 테마를 선택하고 <Visual Studio 시작>을 클릭한다. 

2-4. 잠시 동안 사용준비를 한 후, 처음으로 Visual Studio 화면이 나올 것이다. 오른쪽의 [새 프로젝트 만들

기]를 클릭한다.  

 
 
 
 
 
 
 
 
[그림 8] Visual Studio Community 2022 초기 화면  

2-5. [새 프로젝트 만들기]에서 언어를 'C#'으로 선택한 후, 아래쪽으로 스크롤 해서 'ASP.NET Web Forms 사

이트'를 선택하고 <다음>을 클릭한다. 

[그림 9] Visual Studio Community 2022의 웹 사이트 구축 1  

2-6. [새 프로젝트 구성]에서 나머지는 그대로 두고 [프레임워크]를 '.Net Framework 3.0'으로 선택하고 <만들

기>를 클릭한다. 잠시 후에 자동으로 빈 웹사이트가 구성될 것이다. 

 
 
 
 
 
[그림 10] Visual Studio Community 2022의 웹 사이트 구축 2  

2-7. [웹 폼]이 추가된 프로젝트가 생성되었다.
